# Gestion de Bibliothèque

Projet Java réalisé par Wilson Serefe.

## Fonctionnalités
- Inscription de membres
- Ajout et recherche de livres
- Enregistrement d'emprunts
- Affichage des emprunts en retard

## Prérequis
- Java 21
- Maven
- PostgreSQL (base: bibliotheque, user: postgres)

## Lancement
mvn compile
mvn exec:java
